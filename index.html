Pulse Eco Device

An innovative Arduino-based environmental monitoring device designed for public spaces. The system uses an ESP8266 module to receive air pollution data and displays it through an RGB LED indicator system and a 4-digit 7-segment display. The device is housed in a sleek black box installed at bus stops, providing real-time air quality information to the public. This project aims to raise environmental awareness and provide valuable data about local air quality conditions.
